# 📊 Stock Data Intelligence Dashboard

## 🚀 Objective
This project collects stock data, processes it, and provides insights via APIs and visualizations.

## 🛠 Tech Stack
- Python
- FastAPI
- Pandas
- Matplotlib

## 📌 Features
- Fetch stock data
- Clean & analyze data
- REST API endpoints
- Visualization dashboard

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
