# 📊 Stock Data Intelligence Dashboard

## 🚀 Objective
This project collects stock data, processes it, and provides insights via APIs and visualizations.

## 🛠 Tech Stack
- Python
- FastAPI
- Pandas
- Matplotlib

## ✨ Features
- Stock data fetching (yfinance)
- Data cleaning & analysis
- REST API (FastAPI)
- Interactive dashboard (Chart.js)
- Stock comparison feature
- Volatility / insights

## 📊 Example APIs
- /data/TCS
- /summary/TCS
- /compare?symbol1=TCS&symbol2=INFY


## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
